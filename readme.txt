Categories for media is a Wordpress plugin that allows you to add categories for your media library.

== Description ==

This plugin adds categories for media and allows to sort it by category.

Compatible with Wordpress 3.0+.

= Where I should upload the images that compose this plugin? =

You don't need to upload anything else, all the documents are inside the plugin directory, in the case of the images, this directory is called: 'images'.
